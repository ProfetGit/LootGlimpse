# LootGlimpse

**LootGlimpse** is a modern, lightweight, and visually stunning loot display addon for World of Warcraft. It provides a premium way to see what you've looted without cluttering your chat window.

## Features

- **Visual Excellence**: High-quality animations (Fade, Slide, Pop) and a sleek design that fits right into the modern WoW UI.
- **Native Edit Mode**: Fully integrated with WoW's Edit Mode. Move and resize the loot list just like any other UI element.
- **Customizable**:
  - **Themes**: Choose from Classic (Gradient), Solid, or Minimal.
  - **Fonts**: Customize font face, size, and outline.
  - **Filters**: Filter loot by quality (Poor to Legendary).
  - **Backgrounds**: Optional quality-colored backgrounds.
- **Smart Grouping**: Automatically groups multiple items and updates counts.
- **Interactive**: Hover over items to pause the timer and see the full tooltip.

## Usage

The addon works out of the box! Just loot something and watch it appear.

### Slash Commands

- `/lg` or `/lootglimpse`: Open the settings menu.
- `/lg test`: Spawn a test item to see how it looks.
- `/lg test waterfall`: Spawn a cascade of test items to check animations and layout.

## Configuration

You can access the configuration via the **Options** menu (Game Menu -> Options -> AddOns -> LootGlimpse) or by typing `/lg`.

## Installation

1. Download the latest release.
2. Unzip into your `Interface/AddOns` folder.
3. Ensure the folder is named `LootGlimpse`.
